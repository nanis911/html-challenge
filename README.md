## HTML Challenge
This project involves scraping data from two Mars-related websites and analyzing the information we collect. The goal is to practice web scraping skills and data analysis techniques.

What Was Done:

1. Scraped Mars News Articles:
- Collected the titles and preview text of Mars news articles from a website.
- Stored this information in a structured format.
  
2. Scraped and Analyzed Mars Weather Data:
- Gathered data from a table about Mars weather (e.g., temperatures and pressure).
- Cleaned and analyzed the data to find trends and estimate the length of a Martian year.
  
Websites Used for Scraping:

Mars News Website
Mars Weather Data Website

Tools Used:
- Splinter
- BeautifulSoup
- Pandas
- Matplotlib

Files Included:
part_1_mars_news.ipynb: The notebook where we scraped the Mars news articles.
part_2_mars_weather.ipynb: The notebook where we scraped and analyzed the Mars weather data.
mars_weather_data.csv: The cleaned Mars weather data in CSV format.
